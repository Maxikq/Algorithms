# Kruskal's Algorithm

This implementation finds a weight of Minimum Spanning Tree using disjoint sets.  
http://en.wikipedia.org/wiki/Kruskal%27s_algorithm

## Input
n m  
v1 v2 w	 (m times)  
...  		    
vx vy wz

n - number of vertices  
m - number of edges  
v1 v2 w - edge between "v1" and "v2" with weight "w"

## Sample input
6 10  
1 2 2  
1 6 1  
1 5 3  
4 1 5  
2 6 2  
2 3 5  
4 3 4  
3 5 4  
4 5 4  
5 6 3
